# DomainShark

A simple CLI tool for basic domain intelligence and WHOIS analysis.

## Features

- WHOIS lookup (registrar, dates, status)
- Domain information extraction
- Clean terminal output
- JSON output support

## Requirements

Optional flags:

--json      Output as JSON
--raw       Show raw WHOIS data

- Python 3.10+
- python-whois

## Installation

```bash
pip install python-whois
